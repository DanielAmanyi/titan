# TITAN — Distributed Systems Reliability Testbed

TITAN is a small-scale distributed systems testbed designed to study how distributed workloads behave under infrastructure-level failure conditions.

The system is being developed as an experimental environment to explore questions around fault tolerance, execution semantics, and recovery behavior in distributed task-parallel systems.

## Research Direction

How do distributed systems behave when infrastructure instability is treated as a first-class variable?

## Structure

- architecture/ → system design
- runbooks/ → fault injection protocols
- incidents/ → observed system behavior

## Status

Early-stage testbed under active development.
